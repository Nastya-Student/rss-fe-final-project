# Дата: 2026-03-27

## Что было сделано
На неделе разработал две дополнительные страницы для управления state приложения и получения данных из Supabase на случай изменений в состоянии.
По сути реализовал паттерн наблюдателя (Observer): сделал подписку на изменения событий в user, progress и sessions.
Так же были реализованы 2 дополнительный странциы update and forgot password с восстановление по ссылке на e-mail.
В этот раз не стал заглядывать вперёд — по факту внедрение state завершить не удалось из-за нехватки времени до дедлайна. 
При этом сама архитектура выглядела перспективно и уже напоминала живое приложение. Дополнительно получил полезный опыт работы с базой данных через SQL-запросы.
