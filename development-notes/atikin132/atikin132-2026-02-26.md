## Проблемы

Так как я раньше не использовал JSON файлы напрямую, то не особо понимал, как использовать их в сервисах.

  
## Решения (или попытки)

В итоге почитал, что можно использовать `with { type: "json" }`, и насколько я понял теперь это считается, не исполняемым файлом, а именно JSON.
В итоге получился такой `import`:
```typescript
import  topicsMock  from  "../data/mock-topics.data.json"  with { type: "json" };
```
