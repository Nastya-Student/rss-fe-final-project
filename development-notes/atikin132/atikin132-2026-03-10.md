# Дата: 2026-03-10

## Что было сделано

Сегодня начал разработку Async Sorter Widget. Выбрал этот виджет, потому что хотел научиться работать с Drag and Drop, на курсе уже было задание в котором нужно было это реализовать, но не получилось из-за нехватки времени, было это в RSS Puzzle.

Сделал внешний вид виджета. Показываю пользователю блок кода. В блоке кода есть `console.log` с различнными значениями, к примеру, 1,2,3,4. Так как код приходит из моковых данных в строке с переходом на новую строку `\n`, то делю строку по этому разделителю на массив. Прохожусь по каждому элементу массива и ищу там `console.log`, это нужно, чтобы найти индекс начала того, что выведет `console.log`. Конец ищу начиная с индекса начала и ищу закрывающую скобку. Так получаю то, что выведет `console.log`.

Затем делаю из этих значений отдельные элементы - блоки кода. Также добавил контейнеры с `Call Stack`, `Microtasks`, `Macrotasks`, куда пользователь будет перемещать блоки кода.

Добавил кнопку Run и Output. При нажатии на кнопку Run планирую считывать, что пользователь поместил в контейнеры и выводить это в Output.
